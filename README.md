# Avneet Kumar - Portfolio

This is a simple static portfolio site built with HTML, CSS, and vanilla JavaScript.

## What's included

- Responsive single-page layout (`index.html`)
- Sections based on your CV: Projects, Skills, Education, Certificates, Contact
- Theme toggle (light/dark) persisted in `localStorage`

## How to run locally

You can open `index.html` directly in a browser.

For a local server, you can use any static server, for example:

```powershell
# From this folder
python -m http.server 5500
```

Then open `http://localhost:5500`.

## GitHub Pages (optional)

If you deploy to GitHub Pages, ensure these files are at the repository root:

- `index.html`
- `styles.css`
- `script.js`
